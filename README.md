## Langkah Instalasi Setelah Clone untuk device baru

diterminal ketikan
```bash
pip install requests
 
```
yang kedua 
```bash
pip install lxml
```
yang ketiga
```bash
pip install beautifulsoup4
```

Lalu jalankan aplikasi
